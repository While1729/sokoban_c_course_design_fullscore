# 推箱子小游戏 C语言 README
这是个用C语言写的图形化推箱子小游戏，基于EasyX图形库开发，核心玩法和配套功能都完整跑通了，图片、音乐、音效等素材来源于网络。分享出来给大家做个参考，希望能帮到有需要的朋友~

## 主要功能
- 基础推箱子玩法：WASD控制人物移动，把所有箱子推到目标点就算通关，一共做了10个关卡，难度循序渐进（后几关难度还是较高的）
- 可视化主菜单：鼠标点选就能操作，可选关卡、一键重置全部进度、退出游戏，关卡会按通关状态变色区分，一目了然
- 无限制步数撤销：按空格就能撤回上一步操作，用双向链表做的撤销栈，没有步数上限，不会出现越界闪退的问题
- 自动存档功能：通关进度会自动保存，关掉游戏重开也不会丢失进度，不用从头再打
- 音乐音效配套：带循环背景音乐，还有箱子归位、关卡通关的专属音效，按M键能呼出音乐设置面板，背景音乐和音效的音量、开关都能独立调节
- 实用快捷操作：R键一键重置当前关卡，B键立刻返回主菜单，上手很方便
- 防误触优化：只有游戏窗口在前台激活时才会响应按键，切屏的时候不会误触、乱走

## 运行环境&前置要求
- 系统：Windows（EasyX图形库仅支持Windows系统）
- 编译器：支持EasyX的编译器均可尝试
- 必装依赖：EasyX图形库，不装的话无法正常显示图形界面

## 文件摆放注意事项
所有文件要按下面的结构放，不然会出现找不到贴图、音乐、存档文件的问题，导致黑屏或报错：
```
推箱子游戏/
├── 游戏源码.c
├── images/        // 存放游戏所有贴图素材
├── music/         // 存放背景音乐和音效文件
└── gamesave/      // 存档文件夹，首次运行会自动生成
```

## 基础操作说明
| 按键 | 对应功能 |
|------|----------|
| W/A/S/D | 控制人物上/左/下/右移动 |
| 空格 | 撤销上一步移动操作 |
| R | 重置当前关卡至初始状态 |
| B | 快速返回主菜单 |
| M | 打开/关闭音乐设置面板 |

主菜单界面直接用鼠标左键点击按钮即可，选关卡、重置进度、退出游戏都能直接点。

## 运行步骤
1. 先给你的编译器装好对应版本的EasyX图形库，配置好运行环境
2. 把上面提到的images、music、gamesave三个文件夹，和源码文件放在同一个目录下（一般直接下载压缩不调整即可）
3. 编译或直接运行PushBox.exe即可启动游戏（听说exe 是二进制文件，体积大、每次编译都会变，提交到仓库会把仓库撑得很大，而且别人拉代码会带冗余的二进制文件，不符合代码仓库的规范，所以我删除了）
4. 首次运行会自动创建存档文件，默认关卡为未解锁状态，按顺序通关即可解锁后续关卡

## 一点踩坑提醒
最开始做撤销功能的时候，用了固定长度的结构体数组，玩家操作步数一多就会数组越界，直接闪退。后来换成双向链表动态分配内存，就彻底解决了这个问题，大家做类似功能的时候可以参考这个思路避坑。

整体功能都反复测试过，能正常跑通，没有什么恶性bug。不管是做课设作业，还是初学C语言想做个小项目练手，都可以拿去参考，随便改改都行，希望能帮到大家~
